# Smart Piano User Interface

Smart Piano permet d’apprendre les bases du piano et de perfectionner sa
technique via des exercices "intelligents", s’adaptant aux faiblesses et à
la manière d’apprendre de l’utilisateur.

Ceci est la partie interface utilisateur, communicant avec le moteur.
Elle est spécifiquement développée pour être légère, légère en dépendances et
aisée à embarquer sur une cible ARM 32 bit.
